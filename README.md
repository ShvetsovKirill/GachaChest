# Playable Ads Chest Reward Test

Тестовый проект для демонстрации навыков сборки сцены, UI и VFX в Unity в рамках задач, типичных для playable ads.

---

## 🎯 Цель проекта

Создать небольшой прототип reward-сцены с сундуком, в которой игрок получает награду при взаимодействии с объектом.

Проект демонстрирует навыки:

- сборки сцены в Unity из готовых ассетов;
- настройки освещения, материалов и базовых шейдеров;
- верстки и анимации игрового UI;
- создания VFX с использованием Particle System;
- работы с анимациями;
- организации чистой структуры сцены.

---

## 🧾 Игровая логика сцены

Игрок попадает в небольшое лобби с сундуком награды.

Основное взаимодействие:

1. На экране отображается подсказка **"Tap to get"**
2. Игрок нажимает на экран
3. Сундук немного подпрыгивает
4. Крышка сундука открывается
5. Изнутри появляется свет и вылетают искры
6. Сундук остаётся открытым

---

## 🧩 Реализованные элементы

### 🧱 Сцена

Собрана сцена с использованием предоставленных ассетов:

- сундук с анимациями
- платформа
- текстуры

Настроены:

- освещение
- материалы
- композиция камеры

---

### ✨ Визуальные эффекты (VFX)

Созданы эффекты до открытия сундука:

- искры летящие в сундук для привлечения внимания
- звездочки вокруг сундука
- эффект God Rays

Созданы эффекты открытия сундука:

- свет изнутри сундука
- вспышка при открытии
- искры (Particle System)

---

### 🖥 UI интерфейс

Сверстан UI:

- **Tap to get**
- счетчик монет
- счетчик сундуков
- курсор в виде руки

Также реализована анимация курсора.

В анимации используются:

- scale
- rotation
- position
- alpha

---

### 🎬 Анимация сундука

Настроена последовательность с помощью аниматор контроллера и скриптов:

1. подпрыгивание сундука
2. открытие крышки
3. запуск эффектов
4. переход в состояние открытого сундука

---

## 📱 Особенности playable ads

Проект учитывает ограничения playable ads:

- минимальный вес сцены
- простые VFX
- отсутствие постпроцессинга
- быстрый визуальный фидбек
